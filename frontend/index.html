How It Works

From raw text to explainable predictions in seconds

01
📊

Data Pipeline

Trained on 44,000+ articles from the Kaggle Fake and Real News Dataset. Text is cleaned, tokenized, and encoded using DistilBERT's WordPiece tokenizer.

02
🧠

Model Training

A DistilBERT transformer is fine-tuned for binary classification. The model learns contextual representations to distinguish fake from real news patterns.

03
⚡

Prediction

Articles are passed through the fine-tuned model for real-time inference. The API returns a label (FAKE/REAL) with a calibrated confidence score.

04
🔍

Explainability

SHAP (SHapley Additive exPlanations) and LIME (Local Interpretable Model-agnostic Explanations) highlight which words drove the model's decision.
